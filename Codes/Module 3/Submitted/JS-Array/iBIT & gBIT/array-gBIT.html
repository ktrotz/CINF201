Group Built-It Task (gBIT):

This is part one of a MULTI-PART ASSIGNMENT (with part two occurring in the loops lesson). The ultimate goal is to make a webpage generate a randomized BINGO card. For now, just build (1) a double array that contains these specific values (24 integers and 1 string 'FREE') and (2) use javascript/ECMA to document.write( ) the entire table, filling in the squares from the dobule-array appropriately.
