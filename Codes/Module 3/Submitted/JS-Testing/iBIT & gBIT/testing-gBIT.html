Group Built-It Task (gBIT):

Build a small script that will receive from a user the three coefficients (a,b,c) from a quadratic form (ax^2 + bx + c = 0), and then report to the user the axes of symmetry x = b/(2a) and the approximate values of the roots (note: you will likely need to use Math.sqrt( ); functions to calculate the roots x = (–b ±√(b^2–4ac)) / (2a). For an extra challenge, build this so that the information is written into the body of a document that loads into the browser (as opposed to merely alerting it in pop-up windows).
